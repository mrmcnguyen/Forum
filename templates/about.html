Forums Version 1.0.0

This is the first release of Forums. The functionality of the application is very limited. Users cannot interact with posts by liking or commenting.

All data is stored locally, and no online interaction is possible. The purpose of this application is purely to simulate an online application. At no point in time is this application to be deployed or used to transfer any data over a network connection.
